# URL Shortner

#### environment variables :

```bash
$DBUSR  # Database Username
$DBPWD  # Database Password
$DBNAME # Database Name
$DBADDR # Database Address
$DOMAIN # Server Domain or Address
$LPORT  # Server Port
```

- Install :
```
go build -o Server
```
